##Quickstart

```bash
$ npm install
$ PORT=5000 DATABASE_URL=mongodb://localhost:27017/ccbi-site node start.js
```

##Development

###Bower & Grunt

Install `npm` dependencies:

```bash
$ npm install -d
```

Get vendor libraries using `Bower`:

```bash
$ bower install
```

Watch styles and scripts and build them using `Grunt`:

```bash
$ watch --color -n 1 grunt
```
